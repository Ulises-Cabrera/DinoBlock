Introducción

DinoBlock es una aplicación para dispositivos móviles, la cual tiene un enfoque de programación visual, que permite a los estudiantes desarrollar el pensamiento computacional.


Planteamiento del problema

Aprender los conceptos del pensamiento computacional representa todo un desafío para un estudiante joven sin conocimientos elementales de computación y programación.


Objetivos

Principal Desarrollar una aplicación móvil para apoyar el desarrollo del pensamiento computacional, apoyándonos de tecnologías como lógica difusa, aprendizaje maquina y bajo un entorno de programación visual de Blockly.


Justificación

El desarrollo de esta aplicación de software permitirá que los estudiantes puedan comprender mejor y mas fácil los conceptos del pensamiento computacional, dentro de una plataforma móvil.

Tecnologías utilizadas


FireBase

Firebase es una plataforma para el desarrollo de aplicaciones web y aplicaciones móviles.



Blockly

Blockly es una biblioteca del lado del cliente para el lenguaje de programación JavaScript para crear editores y lenguajes de programación visuales basados ​​en bloques. Es un proyecto de Google y es un software gratuito y de código abierto lanzado bajo la Licencia Apache 2.0.





LibGdx

LibGDX es un framework para el desarrollo de videojuegos multiplataforma, soportando actualmente Windows, Linux, MacOS, Android, IOS y HTML5.





OpenCV

OpenCV es una biblioteca libre de visión artificial originalmente desarrollada por Intel. Open CV es multiplataforma, contiene más de 500 funciones que abarcan una gran gama de áreas en el proceso de visión, como reconocimiento de objetos (reconocimiento facial), calibración de cámaras, visión stereo y visión robótica.





TensorFlow

TensorFlow es una biblioteca de código abierto para aprendizaje automático desarrollada por Google para satisfacer sus necesidades de sistemas capaces de construir y entrenar redes neuronales para detectar y descifrar patrones y correlaciones, análogos al aprendizaje y razonamiento usados por los humanos.





Lógica Difusa

jFuzzyLogic es una biblioteca de lógica difusa de código abierto que implementa estándares de la industria para simplificar el desarrollo de sistemas difusos.
